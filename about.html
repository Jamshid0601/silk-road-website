Project Description

This website explores the ancient Silk Road trade routes through the historical cities of Uzbekistan. It was developed as Coursework 1 for the Essential Web module. The primary goal of this project is to demonstrate a firm understanding of semantic HTML5 structure, modern CSS3 layout techniques (including Grid and Flexbox), and fundamental User Interface (UI) design principles. Furthermore, the site integrates web analytics to monitor user engagement, adhering to modern web development standards.
